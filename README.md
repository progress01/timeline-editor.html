# Report Artifact
基於單一 HTML 檔案的輕量級專案規劃工具。打開瀏覽器即可在電腦、平板或手機上使用。

## 功能

### 1. 雙視圖切換
* **列表模式 (Timeline)**：垂直時間軸，適合手機閱讀與詳細內容編輯。
* **甘特圖模式 (Gantt)**：水平時間軸，適合檢視專案全貌與重疊時段）。

### 2. 支援時間軸配置
* **圖片貼上**：支援 `Ctrl+V` 直接貼上截圖。
* **待辦清單**：
    * **電腦版**：輸入 `[] ` 變 `☐`，輸入 `[x] ` 變 `☑`。
    * **手機/平板**：點擊卡片右上角的 **「☑」按鈕** 快速新增待辦事項。
    * **互動**：點擊方框即可切換完成狀態。
* **日期**：新增節點自動抓取今日日期，並提供「今」按鈕快速設定。

### 3. 匯出與整合
* **AI 輸出**：貼給 ChatGPT/Gemini 幫你寫週報或分析風險。
* **簡報模式**：一鍵隱藏工具列與邊框，介面乾淨，適合投影報告。
* **截圖功能**：自動生成長截圖 (PNG)。
* **CSV 匯出**：可匯入 Google Calendar 或 Excel。

### 4. 資料管理
* **備份/讀取**：支援匯出 `.json` 檔案進行備份，或在不同裝置間轉移資料。
* **空間管理**：提供「清除圖片」按鈕，保留文字但移除圖片以釋放儲存空間。

---

##  流程參考

1.  **開啟**：直接用 Chrome / Edge / Safari 瀏覽器打開 `.html` 檔案。
2.  **新增**：點擊下方的 **「＋ 新增節點」**。
3.  **編輯**：點擊卡片內容即可打字、貼圖。
4.  **工具**：左側（或上方）有便利貼、箭頭等工具可拖拉使用。
5.  **儲存**：系統會**自動儲存**在目前的瀏覽器中（LocalStorage），關掉重開資料還在。

## 注意事項

* **資料儲存**：資料預設存在您的瀏覽器暫存區。**若要清除瀏覽器快取，請務必先點擊「備份」下載 JSON 檔。**
* **圖片限制**：瀏覽器儲存空間有限（約 5MB）。若貼太多圖片導致無法存檔，請使用上方工具列的 **「清圖」** 按鈕釋放空間。
