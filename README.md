# 專案管理工具
基於單一 HTML 檔案的輕量級專案規劃工具。無需安裝、無需登入，打開瀏覽器即可在電腦、平板或手機上使用。https://progress01.github.io/timeline-editor.html

### 1. 多視圖切換
* **列表模式 (Timeline)**：垂直時間軸，適合手機閱讀與詳細內容編輯。
* **大事記模式 (Chronicle)**：自動偵測日期衝突 (⚡)，並顯示工期長度條，適合檢視專案里程碑（手機版採精簡顯示）。
* **甘特圖模式 (Gantt)**：水平時間軸，支援滾輪左右滑動，適合檢視專案全貌與重疊時段。

### 2. 彈性排版與編輯
* **自由縮放 (New)**：
    * **卡片**：滑鼠移至卡片，拖曳右下角「↘」可調整高度與寬度（受中線限制，自動防呆）。
    * **重點框**：紅圈 (⭕) 與藍框 (🟦) 貼紙同樣支援右下角拖曳縮放。
* **圖片貼上**：支援 `Ctrl+V` 直接在卡片內貼上截圖；或直接 `Ctrl+V` 在畫面上產生圖片貼紙。
* **待辦清單**：
    * 點擊卡片右上角的 **「☑」按鈕** 快速插入待辦事項。
    * 點擊內文中的方框 (☐) 即可切換完成狀態 (☑)。
* **日期設定**：新增節點自動抓取今日日期，並提供「今」按鈕快速設定；支援單日或起訖區間。

### 3. 智慧整合與輸出
* **AI 輔助模式**：點擊「🤖 AI」進入選取模式，勾選特定卡片後按「✅ 確認」，自動將內容格式化並複製，方便貼給 ChatGPT/Gemini 撰寫週報或分析風險。
* **簡報模式**：一鍵隱藏所有工具列與編輯按鈕，介面乾淨，適合投影報告（按 ESC 退出）。
* **截圖功能**：自動生成長截圖 (PNG)，需連網載入元件。
* **CSV 匯出**：可匯入 Google Calendar 或 Excel 進行二次運算。

### 4. 資料管理
* **本機儲存**：資料即時儲存在瀏覽器中（LocalStorage），關閉分頁不遺失。
* **備份/讀取**：支援匯出 `.json` 檔案進行備份，或在電腦與手機間轉移資料。
* **空間管理**：提供「🧹 清圖」按鈕，可一鍵移除卡片內的圖片（保留文字與貼紙），釋放瀏覽器儲存空間。

---

##  流程參考

1.  **開啟**：直接用 Chrome / Edge / Safari 瀏覽器打開 `.html` 檔案。
2.  **新增**：點擊下方的 **「＋ 新增節點」**。
3.  **編輯**：點擊卡片內容即可打字、貼圖；使用右下角「↘」調整卡片大小。
4.  **工具**：點擊左側（手機為上方選單）工具箱，拖拉便利貼、箭頭或紅藍框至畫面上。
5.  **儲存**：系統會**自動儲存**，無須手動存檔。

##  注意事項

* **資料安全**：資料存在您的瀏覽器暫存區。**若要清除瀏覽器快取 (Cache)，請務必先點擊「📥 備份」下載 JSON 檔。**
* **圖片限制**：瀏覽器儲存空間有限（通常約 5MB）。若貼太多高解析度截圖導致無法自動存檔（狀態列未顯示「已儲存」），請使用 **「🧹 清圖」** 功能或刪除部分圖片。
